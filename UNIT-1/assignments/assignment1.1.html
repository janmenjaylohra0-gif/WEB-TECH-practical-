HTML Attributes - Explained & Examples

1) What are Attributes?

Attributes are written inside an HTML element and provide additional information about the element — such as name and value (name="value").

2) Example — <a> (href)

This attribute specifies the destination of a link.

Example: Open Wikipedia

Code:

<a href="https://www.wikipedia.org" target="_blank" title="Go to Wikipedia">Open Wikipedia</a>

3) Example — <img> (src, alt)

src defines the path to the image, and alt is shown if the image fails to load.

Example:

Code:

<img src="https://images.unsplash.com/photo-1503023345310-bd7c1de61c7d" alt="Coding Example">

4) Some Other Common Attributes

  • class — Assigns a group or class for CSS or JavaScript.
  • id — Provides a unique identifier for each element.
  • title — Shows a small tooltip when hovered over.
