HTML <img> Tag

1) Function of the <img> Tag

The odede><img> tag in HTML is used to display images on a webpage. It is an empty (self-closing) tag, meaning it doesn’t require a closing tag. The image can be from the same folder or from an external source (URL).

2) Common Attributes and Their Importance

  • src — Specifies the path or URL of the image file that will be displayed on the webpage. Without this, the browser cannot display the image.
  • alt — Provides alternative text that appears if the image fails to load. It helps users understand the content if the image is missing and improves SEO accessibility.

3) Example

example:

Code:

<img src="nature.jpg" alt="nature">
